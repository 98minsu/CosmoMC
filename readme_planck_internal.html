Planck Cosmomc Readme

Compiling

Set highL = ../highL to build direct ACT/SPT support (for use with PLANCKLIKE = cliklike_CamSpec). You can either use clik or other likelihoods directly in an interchangeable way, parameter definitions etc are the same.

Data files

A set of .ini files are supplied in ./batch1 which set up standard for Planck runs. Location of data files is set in CAMspec_ACTSPT_defaults, CAMspec_defaults depending on what you want to use. You can either edit these, or make symbolic links from ./data/clik to the relevant actual file locations depending on what you want to use (and ./data/highL and ./data/camspec if you want to run them outside clik).

Branches

  • final_nominal is the code used for 2013 first papers
  • cosmomc_public is the corresponding public version
  • camspec_wiggle has some camspec likelihood variations etc used for tests
  • tmp_cl_dump has hack to dump C_l when postprocessing (for rainbow plots)
